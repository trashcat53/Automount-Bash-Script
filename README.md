# Automount-Bash-Script
simple bash script for Linux servers to automatically mount selected hard drives, SD cards, flash drives, etc... and add them to fstab. 

I created this script to automate a tedious process for my NAS. This script can also be used on normal linux OS's.

# How to run script:
1. download script or copy contents into sh file
2. sudo chmod +x automnt.sh
3. sudo ./automnt.sh
